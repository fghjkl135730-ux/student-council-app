# 학생회 행사 관리 시스템 - 디자인 가이드

## 앱 개요
학과 학생회를 위한 행사 기획 자동화 앱. 행사 생성 시 부서별 업무 자동 할당, 기획안/사무보고서 템플릿, 시설 대여 관리, 캘린더, 회의록 자동 요약 기능 포함.

## 화면 목록

### 탭 1: 홈 (행사 목록)
- 이번 달 행사 요약 카드 (진행중/예정/완료 수)
- 행사 목록 FlatList (카드형)
  - 행사명, 날짜, 장소
  - 부서별 업무 진행 상태 미니 배지
  - 기획안 작성 / 사무보고서 버튼
- 우상단 + 버튼 → 행사 생성 모달

### 탭 2: 캘린더
- 월별 캘린더 뷰 (react-native-calendars)
- 날짜 탭 시 해당 날짜 행사 목록 표시
- 행사 있는 날짜에 도트 마커

### 탭 3: 시설 대여
- 이번 달 대여 필요 건 요약 테이블
- 장소별 카드 목록
  - 장소명, 목적(행사명), 날짜/시간
  - 대여 신청 / 예약 취소 버튼
  - 상태 배지: 대여 필요 / 신청 완료 / 취소됨

### 탭 4: 더보기
- 회의록 관리 (AI 자동 요약)
- 부서별 업무 현황 전체 보기
- 앱 설정

### 모달/스택 화면
- 행사 생성 화면 (BottomSheet 스타일)
- 행사 상세 화면 (업무 현황, 문서 버튼)
- 기획안 작성 화면
- 사무보고서 작성 화면
- 회의록 작성/요약 화면

## 핵심 사용자 플로우

### 플로우 1: 행사 생성 → 업무 자동 할당
1. 홈 탭 → + 버튼 탭
2. 행사 이름, 날짜, 시간, 장소 입력
3. 스마트 체크리스트에서 필요한 업무 선택
4. "행사 생성" 버튼 → 각 부서에 할 일 자동 생성
5. 장소 대여 체크 시 → 시설 대여 탭에 자동 등록
6. 홈 화면에 새 행사 카드 표시

### 플로우 2: 기획안 작성
1. 행사 카드 → "기획안" 버튼 탭
2. 기획안 템플릿 화면 열림
3. 행사 개요 자동 채워짐
4. 세부 프로그램 및 진행 내용 입력
5. 저장

### 플로우 3: 사무보고서 작성
1. 행사 카드 → "사무보고서" 버튼 탭
2. 구매 물품 리스트 입력
3. 특이사항 입력
4. 증빙사진 체크리스트 확인
5. 저장

### 플로우 4: 회의록 AI 요약
1. 더보기 탭 → 회의록 관리
2. 새 회의록 작성 또는 텍스트 붙여넣기
3. "AI 요약" 버튼 탭
4. 서버 LLM이 핵심 내용, 결정사항, 액션아이템 요약
5. 요약본 저장

## 컴포넌트 구조

```
app/
  (tabs)/
    index.tsx          ← 홈 (행사 목록)
    calendar.tsx       ← 캘린더
    facilities.tsx     ← 시설 대여
    more.tsx           ← 더보기
  events/
    create.tsx         ← 행사 생성
    [id].tsx           ← 행사 상세
    [id]/
      plan.tsx         ← 기획안
      report.tsx       ← 사무보고서
  meetings/
    index.tsx          ← 회의록 목록
    [id].tsx           ← 회의록 상세/요약
components/
  events/
    EventCard.tsx
    TaskBadge.tsx
    DepartmentBadge.tsx
    SmartChecklist.tsx
  documents/
    PlanTemplate.tsx
    ReportTemplate.tsx
  facilities/
    FacilityCard.tsx
    MonthSummary.tsx
  meetings/
    MeetingCard.tsx
lib/
  store/
    events.ts          ← 행사 상태 관리
    tasks.ts           ← 업무 상태 관리
    facilities.ts      ← 시설 대여 상태
    meetings.ts        ← 회의록 상태
  types.ts             ← 공유 타입
  constants.ts         ← 부서, 업무 타입 상수
```
